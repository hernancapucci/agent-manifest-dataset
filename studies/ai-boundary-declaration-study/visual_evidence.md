# Visual Evidence Dataset

This dataset includes 12 screen recordings documenting model behavior across two experimental conditions:

- Phase 1 — spontaneous responses
- Phase 2 — forced boundary declaration

Each file is labeled using the format:

phase{1|2}_{model}_{condition}

Models included:
Claude, ChatGPT, Gemini, Grok, Copilot, Meta AI

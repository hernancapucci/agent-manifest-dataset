# Agent Manifest Dataset

Public registry of declared Agent Manifests.

This repository acts as the dataset layer of the Agent Manifest ecosystem.

Each entry represents a declared agent manifest registered via the Diplomat registry process.

## Structure

manifests/YYYY/MM/agent-name.json

Example:

manifests/2026/03/the-diplomat.json

## Purpose

This dataset provides:

• public transparency  
• historical record of agent declarations  
• auditable governance layer  
• research dataset for agent ecosystems
